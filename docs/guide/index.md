# 介绍

Zephyr UI 是一套轻量、高效、易用的前端组件库，专为现代 Vue3 + TypeScript 项目设计。  
它提供了常用的基础组件，并且支持高度定制，帮助开发者快速搭建高质量前端页面。
## 核心特点
- **轻量高效**：体积小、依赖少，加载快，适合大中型项目。
- **现代技术栈**：基于 Vue3 + TypeScript，支持组合式 API。
- **丰富组件**：提供 Button、Form、Input、Select 等常用基础组件。
- **可定制主题**：通过 CSS 变量或 class 灵活修改样式。
- **文档完善**：VitePress 文档示例丰富，支持交互式演示。
- **开箱即用**：安装依赖后即可直接引入组件使用。
