# ZephyrAudioConverter 处理器

## 简介
你对距离的把控，总是忽远又忽近。

## 场景描述
靠得太近，
细节被无限放大，
呼吸声、摩擦声、环境噪音一并涌入，真实，却让人不适。
离得太远，
方向感消失，层次被压扁，声音只剩下“存在”，却失去了“空间”。
在音频处理的世界里，距离从来不是一个简单的数值，而是一种需要被精确拿捏的平衡。
